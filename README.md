# PRODIGY_TrackCode_02
📌 Description
This project is a simple stopwatch web application built using HTML, CSS, and JavaScript. It allows users to measure time with start, pause, and reset functions. The app also includes a lap feature to record multiple time intervals. This project helps in understanding basic JavaScript concepts like functions, timers, and DOM manipulation.

🚀 Features
Start the stopwatch
Pause the stopwatch
Reset the time
Record lap times
Simple and user-friendly interface

🛠️ Technologies Used
HTML
CSS
JavaScript

▶️ How to Run
Open the project in any browser
Or paste the code into Lovable AI
Click Run / Preview
